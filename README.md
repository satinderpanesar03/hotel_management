# Laravel Vue.js Hotel Management

## Overview

This project is a web application built with Laravel as the backend and Vue.js as the frontend. 

## Features

- User authentication
- CRUD operations
- API integration
- Responsive design using Vue.js and Vuetify

## Technologies Used

- Laravel (PHP)
- Vue.js (JavaScript)
- Vuetify
- Axios (for HTTP requests)
- MySQL (database)

## Installation

- composer install
- npm install
- cp .env.example .env
- php artisan key:generate
- php artisan migrate:fresh --seed
- php artisan storage:link

## Running

- php artisan serve
- npm run dev

### Prerequisites

- PHP >= 8.2
- Composer
- Node.js >= 12
- npm
- MySQL
